# 📚 Book Cover Images Guide

## Folder Structure

Place your cover images in this folder with these exact names:

```
img/covers/
├── adventure-1.jpg
├── adventure-2.jpg
├── fantasy-1.jpg
├── fantasy-2.jpg
├── friendship-1.jpg
├── friendship-2.jpg
├── animals-1.jpg
├── animals-2.jpg
├── space-1.jpg
├── space-2.jpg
├── ocean-1.jpg
└── ocean-2.jpg
```

## Image Requirements

### Recommended Specifications:
- **Format:** JPG or PNG
- **Size:** 800x1000 pixels (portrait orientation)
- **Aspect Ratio:** 4:5 (book cover ratio)
- **File Size:** Under 500KB each
- **Quality:** High resolution, colorful, kid-friendly

### Design Tips:
- Bright, vibrant colors
- Kid-friendly illustrations
- Leave space in center for title text
- Avoid too much text in the image
- Make sure images work with white text overlay

## Themes

### 1. Adventure (2 images)
- Mountains, forests, treasure maps
- Explorers, hiking, camping
- Exciting outdoor scenes

### 2. Fantasy (2 images)
- Castles, dragons, magic
- Wizards, unicorns, enchanted forests
- Magical and mystical scenes

### 3. Friendship (2 images)
- Kids playing together
- Holding hands, sharing
- Happy, warm, friendly scenes

### 4. Animals (2 images)
- Cute animals, pets
- Forest creatures, zoo animals
- Colorful and friendly animal scenes

### 5. Space (2 images)
- Rockets, planets, stars
- Astronauts, aliens
- Cosmic and futuristic scenes

### 6. Ocean (2 images)
- Underwater scenes, fish
- Mermaids, dolphins, sea creatures
- Blue, aquatic, beach scenes

## How It Works

1. System detects story theme
2. Randomly picks 1 of 2 images for that theme
3. Displays as book cover background
4. Adds overlay and title text on top

## Fallback

If images are missing, the system will use beautiful CSS gradient backgrounds as fallback.

## Adding Your Images

1. Save your images with exact names above
2. Place them in this folder (img/covers/)
3. Refresh the page - they'll work automatically!

No code changes needed! 🎉
